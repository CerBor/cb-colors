# What is this?

This is simple library to help you with colors

# Installation

`npm i cb-colors --save`

## Options
```javascript
import cb-colors from 'cb-colors'

console.log(cb.color("purple_heart"));
```

# Documentation
## functions:
```javascript
cb.color();
```
## API:
### Get all color's count:
```javascript
cb.count
```

### Get author:
```javascript
cb.author
```

### Get version:
```javascript
cb.version
```
